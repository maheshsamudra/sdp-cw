## Requirements:

-   PHP version 8 or above
-   My SQL version 5.7 or above
-   Composer

## Setting up the project:

-   composer install
-   run `cp .env.example .env`
-   Create a DB and update the .env file
-   `php artisan key:generate`
-   `php artisan migrate`
-   `php artisan db:seed`
-   `php artisan serve`
